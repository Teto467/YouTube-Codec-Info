# 🎬🔊 YouTube動画の舞台裏を覗き見！コーデック情報表示拡張機能 🎨✨

**「今見ている動画、一体どんな技術で動いているの？」「このヌルヌル動画、本当に60fps？」「音質、これってOpus？AAC？」**

YouTubeを見ている時、ふとそんな技術的な疑問が頭をよぎったことはありませんか？ 🤔 この拡張機能は、そんなあなたの**燃えるような知的好奇心**を満たすために生まれました！ 🎉

この拡張機能をインストールすると、YouTubeの動画プレイヤーの上に、**普段は隠されている動画の「秘密の情報」**がひっそりと表示されるようになります。まるで、動画の技術的な**「成分表示ラベル」**や**「履歴書」**を見ているかのようです！

## これがあなたの新しい「動画探偵グッズ」だ！🕵️‍♂️

プレイヤーの左下あたりに現れる、ちょっと控えめな半透明の箱にご注目ください。そこに表示されるのは、以下のようなマニアックで（でも知るとちょっと嬉しい？）情報です:

*   **🎬 ビデオコーデック:** 動画の圧縮方式（例: avc1, vp9, av01）。「ふむふむ、この動画は最新のAV1か、やるな！」とか、「まだAVC1か、まぁ十分だけどね！」とか、心の中で批評できます。
*   **🖼️ 画質＆フレームレート:** 解像度（例: 1920x1080）や品質ラベル（例: 1080p）と、その動画が1秒間に何枚の絵で構成されているか（例: @60fps）。「なるほど、これが噂のフルHD 60fpsか！私の目はごまかせないぞ！」。
*   **🔊 オーディオコーデック:** 音の圧縮方式（例: mp4a (AAC), opus）。「BGMが心地いいと思ったら、やっぱりOpusか！」とか、「このポッドキャスト、音源はAACだな」とか、耳だけでなく目でも音質を確認？できます。
*   **👂 オーディオ詳細:** さらにディープに、サンプルレート（例: 48kHz）やチャンネル数（例: 2ch）まで！ もうあなたは立派なオーディオマニアの入り口に立っています！。
*   **🎨 色空間情報:** BT.709 / BT.709 / BT.709 とか、謎の呪文のような情報。これは色の規格に関する情報で、ここを理解し始めると、あなたはもう引き返せないマニアックな世界へ足を踏み入れています。
*   **🔴 ストリーム形式:** 今見ているのがリアルタイムな「LIVE」配信なのか、それとも最新技術を使った「DASH」 や「HLS」 といった形式なのか、配信形式まで分かります。

## なぜこの拡張機能が必要かって？ それは...

正直、動画を見るだけなら、これらの情報は**全く必要ありません！** 😂
でも、あなたはきっと、
*   単に技術的なことが気になる、**ちょっと変わった人！**
*   **「最高の画質・音質で再生されてるか確認しないと落ち着かない」というデジタル強迫観念の持ち主！**
*   友達に「この動画、コーデックこれだよ！」と**ドヤ顔で自慢したい！**

そんな、**「見えない情報が見えること自体が楽しい！」**という、**選ばれし賢者（または変態）**のための拡張機能なのです。✨

このオーバーレイは邪魔にならないよう、クリックしても反応しません。動画が変わると一旦情報がリセットされ 、新しい動画の情報が表示されます。フルスクリーン表示時も、ちゃんと邪魔にならない位置に移動しますよ。

## 舞台裏の魔法（ちょっとだけ解説）

この賢い拡張機能は、いくつかのスクリプトが連携して動いています:

*   `manifest.json`: 拡張機能の「身分証明書」兼「取扱説明書」。名前や説明、YouTubeでしか動かないことなどが書いてあります。
*   `content.js`: YouTubeのページ上でこっそり動き回る「エージェント」。情報を表示する箱を作ったり、画面の変化を見張ったり 、そして一番重要なのは... `inject.js` という別のスパイに「頼む！プレイヤーの中の秘密情報を取ってきてくれ！」と依頼する役割です。
*   `inject.js`: こいつが一番のクセモノ！ YouTubeプレイヤーの最も深い部分、普段のウェブページからは触れない「聖域」に潜り込み、プレイヤーオブジェクトの内部情報（`player.getPlayerResponse()` や `window.ytplayer.config.args.player_response` など）を直接引きずり出します。そして、得られたコーデック情報を `content.js` にこっそり報告します。
*   `style.css`: 情報が表示される箱の「見た目担当」。色や形、大きさを整えて、あなたに見やすいようにしてくれます。

これらのファイルが連携することで、あなたの知的好奇心を満たす（そしておそらく誰も気にしない）情報がリアルタイムで表示されるのです。凄いぞ！🎉
